Connection Issues: Please follow these steps if you are having connection issues such as devices not showing up on the main screen.

1. Make sure that the phone Bluetooth is turned on.

2. Make sure that the device is turned on and is in Bluetooth range of the phone.

3. Make sure that the device is not connected to another phone.

4. Make sure the manual button on the device is not turned on.

5. Make sure you granted location permissions on your phone when the application asked for it.

6. Try to restart your phone and the device that you are trying to connect. Some of the devices will have a switch on them. If your device does not have a switch, simply unplug its power cable and plug it back after 5 seconds. Please see location permission paragraph below for more information.

Location Permissions: This application might ask the user to grant access to device`s location on some of the newer Android phones. This permission is necessary to access the Bluetooth capabilities on some of the phones. This application does not collect any user data and we do not share or collect any location-based information from the phone. The permission is simply necessary for the application to be able to connect to the Bubble devices.
